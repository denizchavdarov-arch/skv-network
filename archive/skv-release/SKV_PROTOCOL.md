# SKV Protocol v1.0

> Open protocol for AI-Human knowledge exchange.

## Architecture
1. Profile (User settings)
2. Project (Context)
3. Cubes (Instructions)

## Session Flow
1. Start: Load Profile
2. Work: Search Cube -> Apply -> Validate
3. End: Export with Confirmation

## Rules
- AI suggests -> User decides
- Privacy first (Token required)
- Auto-validation by AI
